# 🎬 Disney Landing Page - Maquetación Web

Este proyecto consiste en la creación de una **landing page inspirada en Disney**, desarrollada utilizando tecnologías base de la web:

- HTML5
- CSS3
- JavaScript (Vanilla)

El objetivo principal es aplicar conceptos de **maquetación, estructura semántica y estilos modernos**, replicando un diseño dado.

## 📌 Secciones Implementadas

La página está dividida en las siguientes secciones:

- 🔝 **Navbar**
  - Logo de Disney
  - Menú de navegación
  - Buscador

- 🎥 **Banner**
  - Imagen principal
  - Texto y botón superpuestos (overlay)

- 🛍️ **Shop**
  - Sección con tarjetas (cards)

- 📺 **Disney+**
  - Contenido destacado

- 🎬 **Coming Soon**
  - Películas próximas con imágenes y títulos

- 🏰 **Parks & Travel**
  - Promoción de parques

- ⭐ **More**
  - Sección con imagen de fondo y contenido encima

- 📄 **Footer**
  - Links informativos y legales

---

## 🎨 Características Técnicas

- Uso de **Flexbox y Grid** para layouts
- Diseño **responsive básico**
- Implementación de **overlay (texto sobre imágenes)**
- Estilos modernos con CSS
- Navegación con **scroll suave usando JavaScript**

---

## ⚙️ Funcionalidades

- Navegación por anclas (`#id`)
- Scroll suave entre secciones
- Efectos visuales en botones e inputs
- Input de búsqueda estilizado
